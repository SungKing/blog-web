vim 使用技巧

1.基本技巧
 vim <file> 打开一个文件，不管文件存在不存在
 vim -O/o <file1>.... 分屏打开文件	
 i 在当前位置进入插入模式， I 在当前行首进入插入模式
 gg 跳转到首行
 ngg n 代表数字，表示跳转到第几行
 G 跳转到文件末尾
 :set nu 显示行号 :set nonu 隐藏行号
 :w  保存。:q 退出。这二者加上!表示强制的意思，vim 里面的!表示的意思不是强制就是相反.
 yy 表示复制本行
 nyy 表示从本行复制之后的多少行
 dd 表示删除本行
 ndd 表示删除本行之后的n行
 p 表示在光标的下一行粘贴。P 表示在光标的上一行粘贴
 J 将本行和下一行合并成一行
 u 复原之前的动作
 ctrl + r 重做上一动作
 . 重复前一动作 

2. 块选择模式
 批量注释
 ctrl+v 进入块选择模式，然后你就知道了...
3. 分屏操作
 :sp file 上下分割
 :vsp file 左右分割（垂直分割）
 ctrl +w + (h j k l ) 向左下上右移动光标至相应的屏幕
 ctrl+w + n(< > + -) n 代表数字，默认1，< 代表宽度缩小，>代表宽度增大，+代表高度增加，- 代表高度降低
 ctrl+w + = 平分屏幕
 :E 浏览目录
 :ls 浏览缓存区的文件
 :buffer n  打开缓存区的文件   
4. 窗口分屏浏览
 :He （hexplorer） 在下边分屏浏览目录
 :He! 在上边分屏浏览目录 
 :Ve （Vexplorer） 在左边分屏浏览目录
 :Ve!  在右边分屏浏览目录
5. 分页浏览目录
 :Te 全称 Texplorer
 gt -到下一页
 gT -到上一页
 ngt -到第n页
6.保存会话
 :mksession ~/.mysession.vim
 :mksession! ~/.mysession.vim 文件名称重复的话，这个强制覆盖
 vim -S ~/.mysession.vim 下一次打开，和上次的一样
7. 保存全部
 :wqa  -保存全部并退出全部
 :qa   – 退出全部
8. other
 v 代表进入选择模式，这时候移动光标可以选择任意东西，然后 y 表示复制，d 表示删除，p 就是粘贴喽
 x 代表剪切
